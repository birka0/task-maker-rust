Crate task_maker_format[−][src]

Task parsing and execution using computation DAGs.

This crate allows you to parse the tasks on disk and evaluate the solutions inside of them by adding the executions inside an ExecutionDAG.

This crate also provides ui functionalities for showing the progress and the results of the execution.
